## Executive summary

The app has a **cohesive dark visual language** (glass panels, sub-pills, accent colors per tab). The **Library Projects/Videos split works** and no longer exposes raw folder names. The biggest problems are **functional** (FreeCut editor shows “Not Found”), **information architecture** (Runs buried in Settings; duplicate navigation labels), and **polish** (duplicate card copy, misleading status colors, persistent HEALTH warning).

## Cross-cutting issues

### 1. Navigation & information architecture 🟠
- Top nav is **6 tabs** (LIVE, PROMPTS, MOTION, MODULATION, AUDIO, SETTINGS). **RUNS** and **GENERATE** are no longer top-level; Runs lives under **SETTINGS → RUNS**.
- Users expecting a dedicated Runs or Generate tab (older docs/tests reference them) will not find them in the main nav.
- **Recommendation:** Restore RUNS as a top-level tab, or add a sticky sub-nav under SETTINGS when active.

### 2. Duplicate edge labels 🟡
- **SETTINGS** appears as: top tab · vertical edge tab on preview · panel context.
- Same pattern on PROMPTS, MOTULATION, AUDIO, MOTION.
- **Recommendation:** Keep one primary label; edge tabs should be icon-only tooltips.

### 3. HEALTH! badge 🟡
- Orange **HEALTH!** shows on every page. Meaning is unclear; reads like an error but may be nominal.
- **Recommendation:** Green “OK” when healthy; orange/red only on degradation, with tooltip.

### 4. Session default name 🟡
- Default session **clown_set_01** appears in chrome on all captures.
- **Recommendation:** Neutral default (`session_01` or date-based).

## Recommended fix order

1. 🔴 **Fix FreeCut embed** — verify `/freecut/` route, iframe `src`, base path, and CSP in production and headless CI.
2. 🟠 **Deforum preview error handling** — timeout overlay on SETTINGS → ENGINE; don’t spin forever when Forge unreachable.
3. 🟠 **Library card copy** — show title once; meta only for secondary info (date, size).
4. 🟠 **Runs discoverability** — top-level RUNS tab or persistent SYSTEM entry from LIVE.
5. 🟠 **Collab WS status colors** — red/grey disconnected, green connected.
6. 🟡 **Motion sequencer copy** — reconcile “No tracks yet” vs pre-defined lanes.
7. 🟡 **LoRA primary buttons** — enable styling when Forge connected; increase contrast when disabled.
8. 🟡 **Audio panel** — remove duplicate header; add upload CTA.
9. 🟡 **HEALTH badge semantics** — only warn when unhealthy.
